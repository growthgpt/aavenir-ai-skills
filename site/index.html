The Ecosystem

Five AI skills.
One Aavenir output contract.

Each skill is a self-contained capability module — auto-triggering on the right kinds of tasks, returning structured JSON that flows straight into Aavenir Contractflow, Obligationflow, RFPflow, Onboardingflow, Invoiceflow, and AVY. Zero translation. Zero glue code.

Powers Contractflow

Contract Intelligence

Reads MSAs, NDAs, SOWs, DPAs, vendor agreements. Extracts parties, dates, values, governing law. Scores risk per clause with evidence-based citations. Flags missing standard clauses.

  • ✓Clause taxonomy
  • ✓Risk scoring
  • ✓Obligation extraction
  • ✓Missing-clause detection
  • ✓Auto-trigger discovery
  • ✓Renewal & termination

The Front Door

AVY — Aavenir Universal Assistant

The single AI front door across Contractflow, Obligationflow, RFPflow, Onboardingflow, and Invoiceflow. Users don't need to know which module owns their question. They ask in natural language — AVY routes, synthesizes across domains, and returns an auditable answer with a structured action log.

  • ✓Multi-domain composition

from agents import Agent, Runner
from pydantic import BaseModel
from pathlib import Path

class ContractReview(BaseModel):
    parties: list[str]
    risks:   list[dict]

agent = Agent(
  name="aavenir-contract",
  instructions=Path(
    "skills/contract-intelligence/prompt.md"
  ).read_text(),
  output_type=ContractReview,
)

result = Runner.run_sync(agent, "…")
